<h1 align="center">
  Stellar Engine
</h1>

A lightweight rendering engine written in Swift

Featuring:
- Physically based rendering
- Image based lighting
- Dynamic reflection
- Procedural skybox

Planning to implement:
- Collision detection
- SSAO
- Global illumination
- Ray tracing (non real time)
